# Задание 3 — Мини-продукт с LLM-аналитикой: AI Data Analyst Agent
**Интеллектуальный агент для анализа данных на основе GigaChat и LangChain с веб-интерфейсом Streamlit**

## Возможности
- *Автоматический анализ данных* 
- *Безопасность* (защита от prompt injection и опасных операций)
- *Анализируйте только необходимое* (возможность дополнительно указать, на что агенту обратить внимание)
- *Веб-интерфейс* (удобный интерфейс на Streamlit)
- *Работа в браузере* (не требуется дополнительная загрузка программ)

## Требования
- Google Colab
- API ключ от GigaChat
- NGROK token
  
## Подготовка перед запуском
1. **Получите API ключ от GigaChat**. Это можно сделать, зарегистрировавшись на сайте https://developers.sber.ru/
2. **Загрузите свой ключ в раздел 'Секреты'🔑 в Google Colab.** В дальнейшем он используется под переменной os.environ['SBER_API_KEY'] с целью безопасности
4. **Получите NGROK токен**. Это можно сделать, зарегистрировавшись на сайте https://ngrok.com
2. **Загрузите свой NGROK токен в раздел 'Секреты'🔑 в Google Colab.** В дальнейшем он используется под переменной NGROK_AUTH_TOKEN с целью безопасности
5. **Установить необходимые библиотеки** Остальные библиотеки уже предустановлены. Некоторые библиотеки из коанды выше тоже уже предустановлены, но для улучшения совместимости их лучше обновить. После установки ячейку можно удалить.

## Запуск кода
1. Откройте  Google Colab (https://colab.research.google.com)
2. Загрузите в Google Colab в одну папку либо файлы 'agent.py', 'app.py' и 'run.py', либо единый файл 'Задание_3_.....' 
3. Запустите код в файле 'run.py', предварительно убрав `%%writefile run.py`, если вы загрузили 3 файла, либо выполнив ячейку с кодом внизу файла 'Задание_3_.....' под заголовком 'run.py'
4. Дождитесь выпонения. В результате выполнения, в выходных данных ячейки появится ссылка на сайт.
   Пример выходных данных (сайт работает в течение нескольких часов после запуска ноутбука в Colab:
   'NgrokTunnel: "https://configure-mud-undress.ngrok-free.dev" -> "http://localhost:8501"'
6. Перейдите на сайт и воспользуйтесь ИИ агентом

## Как пользоваться сайтом**
1. Перейти по ссылке от ngrok
2. Загрузить csv файл, нажав на кнопку 'Upload', расположенную под заголовком "Шаг 1: Загрузите данные"
3. Пропишите свой запрос по анализу данных под заголовком "Шаг 2: Задайте вопрос", при желании вы можете задать дополнительный контекст во второй ячейке этого блока
4. Нажмите на кнопку 'Запустить анализ'
5. Наслаждайтесь готовым анализом
