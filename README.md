# API-пайплайн: Анализатор отзывов с помощью GigaChat
**Автоматический анализ тональности и тематики отзывов с помощью нейросети GigaChat**

Этот код позволяет обрабатывать большие объемы отзывов, определяя тональность отзыва (позитивный/негативный/нейтральный) и тематику каждого отзыва с использованием API GigaChat от Сбера.

## Возможности:
- *Массовая обработка* (обработка большого количества отывов без ручного вмешательства)
- *Определение тональности*
- *Классификация отывов по темам*
- *Повтор запросов* (встроенная система повторов при сбоях API)
- *Экспорт результатов*
- *Работа в браузере* (не требуется дополнительная загрузка программ

## Подготовка перед запуском
1. **Загрузите на свой Google Drive файл с датасетом**, в котором содержатся отзывы. Проследите, чтобы отзывы были размещены в колонке 'Review'. В ином случае, выпадет ошибка
2. **Получите API ключ от GigaChat**. Это можно сделать, зарегистрировавшись на сайте https://developers.sber.ru/
3. **Загрузите свой ключ в раздел 'Секреты'🔑 в Google Colab.** В дальнейшем он используется под переменной key с целью безопасности
4. **Установить библиотеку GigaChat** Хотя удобство Google Colab заключается в уже досупных базовых библиотеках без дополнительной загрузки. Некоторые библиотеки необходимо дополнительно установить. Для установки необходимо ввести указанную ниже команду в кодовое окно коллаба. После установки, это окно можно удалить
   ```python
   pip install GigaChat
**Программа готова к запуску**
