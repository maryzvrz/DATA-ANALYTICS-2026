# API-пайплайн: Анализатор отзывов с помощью GigaChat
**Автоматический анализ тональности и тематики отзывов с помощью нейросети GigaChat**

Этот код позволяет обрабатывать большие объемы отзывов, определяя тональность отзыва (позитивный/негативный/нейтральный) и тематику каждого отзыва с использованием API GigaChat от Сбера.

## Возможности:
- *Массовая обработка* (обработка большого количества отывов без ручного вмешательства)
- *Определение тональности*
- *Классификация отывов по темам*
- *Повтор запросов* (встроенная система повторов при сбоях API)
- *Экспорт результатов*
- *Работа в браузере* (не требуется дополнительная загрузка программ
