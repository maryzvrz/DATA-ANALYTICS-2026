# API-пайплайн: Анализатор отзывов с помощью GigaChat
**Автоматический анализ тональности и тематики отзывов с помощью нейросети GigaChat**

Этот  скрипт читает отзывы → LLM определяет тональность (positive/negative/neutral) и тему → результат в JSON и CSV с использованием API GigaChat от Сбера.

## Возможности
- *Массовая обработка* (обработка большого количества отзывов без ручного вмешательства)
- *Определение тональности*
- *Классификация отзывов по темам*
- *Повтор запросов* (встроенная система повторов при сбоях API)
- *Экспорт результатов*
- *Работа в браузере* (не требуется дополнительная загрузка программ

## Требования
- Google Colab
- API ключ от GigaChat
- Датасет с отзывами в колонке 'Review'
  
## Подготовка перед запуском
1. **Загрузите на свой Google Drive файл с датасетом**, в котором содержатся отзывы. Проследите, чтобы отзывы были размещены в колонке 'Review'. В ином случае, выпадет ошибка
2. **Получите API ключ от GigaChat**. Это можно сделать, зарегистрировавшись на сайте https://developers.sber.ru/
3. **Загрузите свой ключ в раздел 'Секреты'🔑 в Google Colab.** В дальнейшем он используется под переменной key с целью безопасности
4. **Установить библиотеку GigaChat** Остальные библиотеки уже предустановлены. После установки ячейку можно удалить.
   ```python
   pip install GigaChat
    ```
**Программа готова к запуску**

## Запуск кода
1. Откройте  Google Colab (https://colab.research.google.com)
2. Загрузите файл с кодом
3. Последовательно выполните все ячейки
4. Дождитесь завершения 
5. Скачайте `results.csv` и `results.json`

## Настройки внутри кода
- *Укажите путь к файлу с базой данных в Google Drive*
   ```python
   drive.mount('/content/drive')
   csv_path = '/content/drive/MyDrive/data-zav2.csv' -> указать путь здесь
    ```
- *Можно настроить количество повторов запросов под себя*
   ```python
   @retry(
   stop=stop_after_attempt(3), -> число попыток
   wait=wait_exponential(multiplier=1, min=2, max=10) -> задержка между попытками
   )
    ```
- *Настройте частоту запросов*
   ```python
   time.sleep(0.5)
    ```
- **Примечание** - модель GigaChat уже настроена. `verify_ssl_certs=False` используются ввиду проблем с сертификатами GigaChat
